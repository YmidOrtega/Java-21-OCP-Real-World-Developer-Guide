### Estudiando para el examen

Recomiendo firmemente que debe escribir y ejecutar programas como parte del proceso de estudio. La forma de estudiar puede ser tan importante como lo que se estudia.

#### Creando un Plan de Estudio

Roma no se construyó en un día, así que no deberías intentar estudiar para el examen en un solo día. Incluso si has sido certificado en una versión anterior, la nueva prueba incluye características únicas de Java 18-21. Recomiendo que apartes una cantidad de tiempo cada día, aunque sean solo unos minutos, para leer o practicar. La idea es mantener tu impulso. Trata de evitar tomarte unos días o semanas libres de estudiar, o es probable que pases mucho tiempo leyendo el mismo material.

#### Creando y Ejecutando el Código

Aunque algunas personas pueden aprender Java simplemente leyendo un libro de texto, no es así como recomiendo que estudies para un examen de certificación. Quiero que escribas tus propias aplicaciones de Java a lo largo de este proyecto para que no solo aprendas el material, sino que también lo comprendas. Por ejemplo, puede que no resulte obvio por qué la siguiente línea de código no se compila, pero si intentas compilarla tú mismo, el compilador de Java te indicará el problema:

```java
float value = 102.0; // DOES NOT COMPILE
```

#### Software IDE

Mientras estudias, deberías desarrollar código usando un editor de texto y el compilador de línea de comandos. Un IDE (como Eclipse, IntelliJ o Visual Studio Code) puede interferir con tu capacidad para detectar problemas fácilmente en un examen.

#### Identificando tu Eslabón Más Débil

Para cada capítulo, debes anotar qué preguntas respondiste mal, entender por qué, y estudiar esas áreas aún más. Puedes tomar las preguntas de repaso una y otra vez para reforzar tu aprendizaje, siempre y cuando te expliques a ti mismo por qué cada respuesta es correcta.

#### "Sobreestudiar" los Exámenes de Práctica 

Rehacer los exámenes de práctica una y otra vez puede tener un impacto negativo a largo plazo. Algunas personas memorizan las respuestas y se vuelven demasiado confiadas.
