El capítulo anterior trató extensamente sobre la definición de variables, pero ¿qué se puede hacer con una variable una vez que ha sido creada? Este capítulo introduce los operadores y muestra cómo se pueden utilizar para combinar variables existentes y crear nuevos valores. Se muestra cómo aplicar operadores a varios tipos de datos primitivos, incluyendo la introducción a operadores que pueden aplicarse a objetos.
